    ✨项目简介
本仓库是一套面向安卓平台 Minecraft Java 版启动器的完整技术文档合集，收录各类移动端启动器底层原理、图形转译层、Vulkan渲染驱动、JVM运行配置、模组兼容、崩溃报错排查等深度笔记。
仓库内文档分为中文原版与英文翻译版，适合MC启动器开发者、技术爱好者查阅学习，梳理移动端运行Java版MC的各类实现逻辑与疑难问题解决方案。

    🚀 核心内容
 主流安卓MC启动器架构解析：PojavLauncher / FoldCraftLauncher / HMCL移动端等
 图形转译技术：GL4ES、Zink、VirGL、ANGLE等渲染层原理与适配方案
 JVM多版本配置、内存调优、Java环境导入教程
 各大模组加载器移动端兼容记录：Forge / NeoForge / Fabric / Quilt / OptiFine
 闪退、异常报错（XXXException）成因分析与修复方案

    ✨文档亮点
